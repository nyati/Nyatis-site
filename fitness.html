🔗 Google Sheets Setup

Follow these steps to create a free Google Sheets tracker that receives your weekly data:

  1. Go to sheets.google.com → create a new sheet titled "Fitness Tracker"
  2. Add headers in Row 1: Week, Date, Weight, Waist, BF%, Energy, Day, Workout, Exercises, Notes
  3. Go to Extensions → Apps Script
  4. Delete all existing code and paste this:
function doPost(e){var ss=SpreadsheetApp.getActiveSpreadsheet();var sh=ss.getActiveSheet();var d=JSON.parse(e.postData.contents);sh.appendRow([d.week,d.date,d.weight,d.waist,d.bf,d.energy,d.day,d.workout,d.exercises,d.notes]);return ContentService.createTextOutput("OK");}
  1. Click Deploy → New Deployment (top right)
  2. Click "Select type" → Web app (this step is required to reveal the settings)
  3. Set Execute as: Me and Who has access: Anyone
  4. Click Deploy → Google will ask you to review permissions → click Authorize access, choose your Google account, and click Allow
  5. Copy the Web App URL (starts with https://script.google.com/macros/s/…) and paste it above

⚠️ If your Google Workspace account doesn't allow "Anyone", try "Anyone with Google account" instead — you'll need to be signed in to the same browser.

✓ Data saves locally on this device even without Sheets setup.
